⚙️ Configuration

🛠 Setup Guide — Free Hosting on Railway + cron-job.org

1
Fill in your .env file — copy .env.example to .env and add your GO API key, Twilio credentials, wife's phone, and a CRON_SECRET (any random string like bronte-union-2024).
2
Deploy to Railway (free) — push this folder to GitHub, then go to railway.app → New Project → Deploy from GitHub. In the Railway dashboard, go to Variables and paste in all your .env values. Railway gives you a public URL like https://go-notifier.up.railway.app.
3
Set up cron-job.org (free) — sign up at cron-job.org → Create Cronjob.
• URL: https://YOUR-RAILWAY-URL/api/trigger?key=YOUR_CRON_SECRET
• Schedule: Every 2 min · Hours: 7,8 · Days: Mon–Fri
• Method: POST
This pings your server every 2 minutes on weekday mornings, waking it up even if it's sleeping.
4
Test it — click Send Test SMS above to confirm your wife gets a message. Click Check Now to do a full live train status check anytime.
