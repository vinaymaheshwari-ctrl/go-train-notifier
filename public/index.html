🛠 Setup — Railway + cron-job.org (Free)

1
Fill in .env — GO API key (from Metrolinx), Twilio credentials, wife's number, and a CRON_SECRET.
2
Deploy to Railway — push to GitHub, connect repo, paste all .env values into Railway Variables tab.
3
Set up cron-job.org — POST to /api/trigger?key=YOUR_SECRET every 2 min, Hours 7–8, Mon–Fri.
4
Test — click Test SMS to confirm your wife gets the message.
